# Realtime Flood Monitoring 

##  Overview
A system where people can rely on during flood disasters

##  Components
- Arduino Uno
- ESP8266
- Ultrasonic Sensor
- Jumper Wires

##  Features
- Live Viewing of Flood Level through Web
- SMS notification
- User-friendly UI
- Realtime Data

##  Challenges I Solved
- Ultrasonic Sensor is distance sensor which means that it measures the distance from the sensor, so I made my calculation that it will measure backwards to get the water level.
- Arduino and ESP serial communication. I had hard time fixing the serial communication for it is not in the scope of my BSIT course then what I did is I maximize the use of AI like chatgpt, google search and reddit to explore and self study until I fixed the issue.
